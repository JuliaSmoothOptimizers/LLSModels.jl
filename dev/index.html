LLSModels.jl documentation

This package provides a linear least squares model implementing the NLPModels API.

Install

Install LLSModels.jl with the following command.

pkg> add LLSModels

Usage

This package defines LLSModel.

LLSModels.LLSModel — Type
nls = LLSModel(A, b; lvar, uvar, C, lcon, ucon)

Creates a Linear Least Squares model $\tfrac{1}{2}\|Ax - b\|^2$ with optional bounds lvar ≦ x ≦ uvar and optional linear constraints lcon ≦ Cx ≦ ucon. This problem is a nonlinear least-squares problem with residual given by $F(x) = Ax - b$.

source

We can define a linear least squares by passing the matrices that define the problem

\[\begin{aligned} \min \quad & \tfrac{1}{2}\|Ax - b\|^2 \\ & c_L \leq Cx \leq c_U \\ & \ell \leq x \leq u. \end{aligned}\]

A = rand(10, 3)
b = rand(10)
C = rand(2, 3)
nls = LLSModel(A, b, C=C, lcon=zeros(2), ucon=zeros(2), lvar=-ones(3), uvar=ones(3))
